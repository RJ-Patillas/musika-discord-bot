# Musika

Musika is a Discord bot that follows a user's currently playing Spotify track
and automatically searches for a matching version to play in a Discord voice channel.

## Features

- 🎵 Follow Spotify "Now Playing"
- 🔍 Smart song search
- 🔊 Voice channel playback
- 📃 Queue management
- ⏯ Auto pause/resume
- 👥 Multi-user support (planned)

## Tech Stack

- Node.js
- Discord.js
- Spotify Web API
- Git
- GitHub

## Installation

```bash
npm install
```
